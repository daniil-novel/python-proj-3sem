Переменная __all__ - это список имён, которые должны быть доступны для импорта через звёздочку. Если эта переменная
определена в модуле, то при импорте через знак * будут импортироваться только те имена, которые указаны в списке __all__.
Если переменная __all__ не определена в модуле, то при импорте через знак * будут импортироваться все имена, начинающиеся
не с символа подчеркивания.

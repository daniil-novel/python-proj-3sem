Этот код изменит значение глобальной переменной GLOBAL_VAR в текущем модуле, но не отразится на других модулях, которые
импортируют GLOBAL_VAR из some_module. В других модулях значение GLOBAL_VAR останется неизменным.

Чтобы изменить значение глобальной переменной модуля и сделать это изменение доступным для всех модулей, следует изменять
значение напрямую в модуле, а не в импортированных переменных.
